python -m venv .venv - создание виртуального окружения.

pip install Django - установка Django в ваше виртуальное окружение.

django-admin - просмотр команд Django.

django-admin startproject name - создание вашего проекта Django. Вместо name нужно указать имя вашего
приложения.

python manage.py runserver - стартует наш сервер (приложение).

python manage.py startapp name - создаёт новое приложение. Вместо name нужно указать имя вашего приложения.

python manage.py makemigrations - создать миграции.

python manage.py sqlmigrate news 0001

python manage.py migrate - провести миграцию.


Задание 1
Используя механизм шаблонов создайте приложение, отображающее надпись «Hello, World!».

Задание 2
Добавьте к первому заданию поддержку вывода надписи «Hello, World» на немецком, французском, испанском. 
Доступ к надписям на разных языках должен происходить по разным маршрутам. Например:
■ 127.0.0.1:8000 — отображается страница на английском языке;
■ 127.0.0.1:8000/fr — французский;
■ 127.0.0.1:8000/de — немецкий;
■ 127.0.0.1:8000/es — испанский.

Задание 3
Используя механизм шаблонов создайте веб-приложение, отображающее текущее время и дату. В зависимости от времени суток нужно выводить надписи:
■ Доброе утро!
■ Добрый день!
■ Добрый вечер!

Задание 4
Создайте модель "Человек" со всеми необходимыми на вашь взгляд полями.

Задание 5
Создайте форму для ввода имени и фамилии. 
После заполнения и отправки данных необходимо отобразить приветствие: «Добрый день, [Имя Фамилия]!».
Так же необходимо сохранить данные в БД.

Задание 6
Пользователь вводит в текстовые поля в форме начало и конец диапазона чисел. 
Необходимо отобразить все простые числа в этом диапазоне.

Задание 7
Создайте форму для подписки на новости вашей компании. 
Пользователь должен ввести следующие 
данные:
■ Имя;
■ Фамилия;
■ Возраст;
■ Пол;
■ Email.
Полученные от пользователя данные нужно отобразить в браузере.

Задание 8
Пользователь вводит начало и конец диапазона для таблицы умножения. 
Необходимо отобразить всю таблицу в указанном диапазоне.
